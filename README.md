# lulu-exercise

## 构建流程

1. npm init根据提示生成默认node项目及package.json描述文件

2. npm install webpack --save-dev

3. npm i vue --save-dev

4. npm i vue-loader --save-dev

5. npm i webpack-dev-server --save-dev

6. 

## 构建工具

- node v7.9.0，版本较新，可以安装n或者nvm切换不同版本

- webpack v2.4.1
